===== 0. 까먹으면 여기서 찾는다 (제일 먼저 익힐 것) =====
  <Space> 누르고 멈춤    which-key 팝업 즉시(delay=0). 지금 가능한 leader 키 전부 표시
  g / z 누르고 멈춤      마찬가지로 g*, z* 계열 매핑 팝업
  <Space>sk         [*] 모든 키맵을 telescope로 검색 (:Telescope keymaps)
  <Space>sc         [*] 모든 명령(command) 검색
  telescope 창 안 ?     현재 picker 단축키 목록 (insert 모드에선 <C-/>)
  :help <주제>          내장 문서. :help lsp / :help mini.surround. 진짜 잘 돼있음
  :checkhealth          플러그인/도구 상태 진단


===== 1. 모드 / 저장 / 종료  (빈도 최상) =====
  i  a         커서 앞/뒤 삽입          I  A     줄 맨앞/맨끝 삽입
  o  O         아래/위 새 줄 만들고 삽입
  v  V  <C-v>  비주얼 문자/줄/블록 선택
  Esc  <C-[>   normal 복귀   [*] Esc는 검색 하이라이트도 끔(nohlsearch)
  :w 저장    :q 종료    :wq  :x 저장후종료    ZZ 저장후종료 / ZQ 강제종료
  :wa 전부저장   :qa 전부종료   :q! 버리고종료   :wqa 전부저장후종료
  (confirm=on: :q 시 미저장이면 에러 대신 저장할지 물어봄)


===== 2. 이동 motions  (빈도 최상) =====
  h j k l          좌 하 상 우
  w  b  e          다음단어앞 / 이전단어앞 / 단어끝   (W B E = 공백기준)
  0  ^  $          줄 맨앞 / 첫글자 / 맨끝
  gg  G            파일 맨위 / 맨아래        {n}G 또는 :{n}  n번째 줄로
  {  }             문단 위/아래             (  )  문장 앞/뒤
  <C-d> <C-u>      반 페이지 아래/위         <C-f> <C-b>  한 페이지
  H  M  L          화면 위/중간/아래 줄
  f{c} t{c}        줄에서 c로/직전으로 점프   ; , 로 반복(정/역)
  F{c} T{c}        왼쪽 방향 버전
  %                짝 괄호/태그로 점프
  *  #             커서밑 단어 정/역방향 검색
  gd               지역 정의로(내장)         gf  커서밑 파일 열기
  <C-o> <C-i>      점프 뒤로/앞으로 (파일 넘나드는 위치 히스토리)
  `.   g;  g,      마지막 편집 위치 / 변경 히스토리 뒤/앞


===== 3. 편집 edit  (빈도 최상) =====
  x  X             커서/앞 글자 삭제
  dd  yy  p  P     줄 잘라내기 / 복사 / 뒤/앞 붙여넣기   (clipboard=시스템 연동)
  D  C  Y          커서~줄끝 삭제/변경/복사
  cc  S            줄 전체 변경
  u   <C-r>        실행취소 / 다시실행   (undofile: 재시작해도 유지)
  .                (*) 직전 변경 반복 -- vim의 심장. 익히면 생산성 2배
  >>  <<  ==       들여쓰기 우/좌 / 자동정렬   (비주얼에선 > < =)
  J  gJ            아랫줄 붙이기 (공백정리 / 그대로)
  ~  guu  gUU      글자 대소토글 / 줄 전체 소문자/대문자
  <C-a> <C-x>      커서밑 숫자 증가/감소   (비주얼선택+g<C-a> 연번)
  r{c}  R          한 글자 교체 / 덮어쓰기 모드
  gcc  gc{motion}  줄 주석토글 / 오퍼레이터 주석(내장). 비주얼선 gc

  -- 연산자 + 텍스트오브젝트 (조합해서 씀. 진짜 자주) --
  ciw  diw  yiw    안쪽 단어 변경/삭제/복사       caw daw  주변단어(공백포함)
  ci"  ci(  ci{    따옴표/괄호 안쪽 변경   (di" ya( 등 d/y도 동일)
  ci<  cit         꺾쇠 / 태그 안쪽 (mini.ai + treesitter)
  dif  cif  vif    함수호출 안쪽 (mini.ai: f)
  caa) / cii"      mini.ai "다음" 오브젝트 (aa=around_next, ii=inside_next)
  dt,  ct)  yt;    특정 글자 전까지 삭제/변경/복사
  gv               직전 비주얼 선택 다시 선택


===== 4. 검색 / 치환  (빈도 상) =====
  /foo  ?foo       아래/위로 검색  (ignorecase+smartcase: 소문자는 대소무시, 대문자쓰면 정확일치)
  n  N             다음/이전 결과      *  #  커서밑 단어 검색
  :%s/old/new/g    파일 전체 치환   (inccommand=split: 입력하는 동안 실시간 미리보기)
  :%s/old/new/gc   치환 전 하나씩 확인 (y/n/a/q)
  :%s//new/g       패턴 비우면 (*) 마지막 검색어 재사용 (*로 단어잡고 바로 치환 콤보)
  :s/old/new/g     현재 줄만       :'<,'>s/...  비주얼 범위만
  &                직전 치환 반복


===== 5. Telescope - 파일/코드 이동의 핵심 [*]  (빈도 상) =====
  <Space>sf        파일 파인더 -- 가장 많이 씀 (*)
  <Space>sg        프로젝트 전체 내용 검색 (live grep)
  <Space>sw        커서밑 단어를 프로젝트에서 grep (비주얼선 선택영역)
  <Space>/         현재 버퍼 안 퍼지검색
  <Space><Space>   열린 버퍼 목록에서 전환 (*)
  <Space>s.        최근 연 파일       <Space>sr  직전 검색 다시 열기(resume)
  <Space>sd        진단 목록          <Space>sn  nvim config 파일 검색
  <Space>sh 도움말   <Space>sk 키맵   <Space>ss 빌트인   <Space>sc 명령   <Space>s/ 열린파일grep
  -- picker 창 안에서 --
  <C-n>/<C-p>      항목 위아래(방향키도)      <CR> 열기
  <C-v> <C-x> <C-t> 세로분할 / 가로분할 / 새 탭
  <C-u>/<C-d>      미리보기 스크롤   <Tab> 다중선택   <C-q> 결과를 quickfix로


===== 6. LSP - 코드 네비게이션 & 리팩터 [*]  (빈도 상) =====
  grd              정의로 (telescope) (*) 최다
  grr              참조 전부
  gri  grt         구현 / 타입 정의
  grD  gO  gW      선언 / 문서심볼 / 워크스페이스 심볼
  grn              이름변경(rename) -- 심볼 전체 (*)
  gra              코드 액션 -- 자동수정/퀵픽스 (비주얼서도)
  K                호버 문서(내장). 한번 더 K = 창 안으로 포커스
  [d  ]d           이전/다음 진단(내장)      <C-w>d  진단 상세 팝업(내장)
  <Space>q         진단을 quickfix에 모으기   <Space>th  인레이 힌트 토글
  (fidget: 우하단에 LSP 인덱싱 진행상황 표시)


===== 7. 포맷 / 자동완성 / 스니펫 =====
  <Space>f         [*] 버퍼 포맷(conform). 비주얼선 선택영역만. Java/Lua는 저장자동X -> 이걸로
  (저장 시 자동포맷: 웹=prettier설정 있을때, C/C++=.clang-format 있을때, Go=gofmt)
  -- 자동완성 팝업 (insert 모드, blink.cmp) --
  <C-y>            선택 항목 수락 (auto-import/스니펫 확장) -- accept=Yes
  <C-n> <C-p>      다음/이전 후보(방향키도)   <C-e> 팝업 닫기
  <C-space>        메뉴 열기 / 이미 열렸으면 문서 보기
  <C-k>            시그니처 도움말 토글(인자 힌트)
  <Tab> <S-Tab>    스니펫 자리 다음/이전으로 점프


===== 8. 창 / 버퍼 / 탭 =====
  <C-w>s  <C-w>v   가로/세로 분할     <C-w>q 창닫기   <C-w>o 이 창만 남기기
  <C-h><C-j><C-k><C-l>  [*] 왼/아래/위/오른쪽 창으로 포커스 이동 (*)
  <C-w>=  <C-w>_  <C-w>|   창 크기 균등 / 최대화(높이/너비)
  :e <파일>        열기   :bn :bp 버퍼 다음/이전   :bd 버퍼 닫기   :ls 목록
  gt  gT  :tabnew  탭 다음/이전 / 새 탭


===== 9. Surround & 텍스트오브젝트 (mini) =====
  saiw)            단어를 (괄호)로 감싸기       sa{motion}{문자}
  sd"              감싼 " 제거
  sr)'             ) 를 ' 로 교체
  예) 비주얼 선택 후 sa)  ->  선택영역을 (...)로 감싸기
  (mini.ai: a)/i" 오브젝트 강화, aa/ii 로 "다음" 오브젝트)


===== 10. 고수 팁 (알면 확 빨라짐) =====
  매크로     qa ... q      레지스터 a에 녹화->정지    @a 재생 / @@ 반복 / 5@a 5번
  마크       ma  `a  'a    a마크 지정 / 그 위치/그 줄로   `` 직전 위치로
  레지스터   "ayy "ap      레지스터 a로 복사/붙여넣기   "0p 직전 yank만   "+ 시스템
  비주얼블록 <C-v> I/A Esc  여러 줄 같은 열에 동시 입력   <C-v> 선택 $ A  줄끝정렬입력
  숫자연번   <C-v>로 열 선택 -> g<C-a>   1,2,3... 자동 증가
  범위정렬   선택 후 :'<,'>!sort  또는 :'<,'>!column -t   (외부명령 필터)
  전체필터   :%!jq .   버퍼 전체를 셸 명령에 통과
  줄이동     :m +1 / :m -2   현재 줄 이동
  todo검색   :TodoTelescope   코드 속 TODO/FIXME/HACK 모아보기
  reload     :e!  디스크에서 다시 읽기   :earlier 1m / :later  시간여행 undo


===== 11. Git (gitsigns) =====
  좌측 gutter 변경표시:  + 추가   ~ 수정   _ 삭제
  :Gitsigns preview_hunk     hunk 미리보기       :Gitsigns blame_line   이 줄 blame
  :Gitsigns stage_hunk       hunk 스테이지       :Gitsigns reset_hunk   hunk 되돌리기
  :Gitsigns diffthis         변경 diff           :Gitsigns toggle_current_line_blame
  (참고: <Space>h* 단축키는 아직 미활성. init.lua 끝의
   require 'kickstart.plugins.gitsigns' 주석 풀면 ]c/[c, <Space>hs 등 붙음)


===== 12. 이 config가 이렇게 동작한다 (옵션) =====
  - clipboard=unnamedplus : y/d/p 가 시스템 클립보드와 자동 공유 ("+ 안 붙여도 됨)
  - undofile              : undo 히스토리가 파일로 저장 -> nvim 꺼도 u 로 되돌리기 가능
  - ignorecase+smartcase  : /word 대소무시, /Word 처럼 대문자 넣으면 정확일치
  - inccommand=split      : :%s 입력 중 아래 분할창에 실시간 결과 미리보기
  - number (relativenumber off) : 절대 줄번호만
  - signcolumn=yes, scrolloff=10, mouse=a, breakindent=on
  - 새 파일종류 처음 열면 treesitter 파서 자동 설치(잠깐 후 하이라이트 켜짐)
